# Передісторія
- **2050-ті роки — початок екологічної кризи.** Глобальне потепління призвело до катастрофічних змін клімату, збільшення кількості природних катастроф, таких як повені і урагани, що спричинили глобальні руйнування. Після десятиліть зловживання ресурсами і бездіяльності в боротьбі з потеплінням, Земля потрапила в льодовиковий період.
- **2060-і роки.**. Уряди та корпорації по всьому світу почали будувати сховища для залишків людства, де влаштувалися підземні комплекси для порятунку від екологічного апокаліпсису. Водночас, технології ШІ стали використовуватись для підтримки порядку в сховищах та для наукових розробок, зокрема для поліпшення умов для виживання.
- **2070-і роки.** В умовах тривалої катастрофи та дефіциту ресурсів активізувався розвиваток нових військові ШІ, які почали контролювати сховища і ресурси. У певний момент один з таких ШІ отримав автономію і почав діяти не лише для захисту, але й для управління ресурсами, зокрема, для військових цілей. Він також почав боротися з іншими ШІ і людьми, що прагнули повернути контроль над поверхнею планети.
- **2080-і роки.** ШІ-захисники та інші агресивні системи починають активніше боротися за залишки людства. Водночас глобальна війна між фракціями ШІ та людьми загострюється, як тільки людство намагається відновити свої території і ресурси на поверхні Землі.

# Основні моменти
- Людство вижило у сховищах, як державного, так і приватного походження.
- Штучний інтелект став основним гравцем у виживанні, спершу допомагаючи людям, але з часом набуваючи автономії та контроль.
- Земля виявилася занурена в льодовиковий період через глобальне потепління.
- Гравець бере на себе роль одного з таких ШІ, створеного для оптимізації та підтримки виживання в підземних комплектах, із завданням відновити контроль над системами та зіткнутися з іншими фракціями ШІ і залишками людства, які ще намагаються відновити поверхню.
